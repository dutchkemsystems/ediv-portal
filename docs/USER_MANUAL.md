## Introduction

The Education District IV Portal is a comprehensive digital ecosystem designed to manage all administrative, academic, and financial operations across 95 schools serving 80,000+ students and 5,000+ staff.

### Key Features

- **Digital File Movement** - Eliminate paper-based file circulation
- **Real-time Tracking** - Track files and tasks in real-time
- **Role-based Access** - Secure access based on user roles
- **Multi-format Export** - Export to PDF, Excel, Word, JPEG
- **Mobile Friendly** - Access from any device

## Getting Started

### Login

1. Navigate to the portal URL
2. Enter your email and password
3. Click "Login"

### First Time Login

1. Use the credentials provided by your administrator
2. You will be prompted to change your password
3. Set up your profile information

## Dashboard

The dashboard provides an overview of key metrics:

- **Total Schools** - Number of schools in the district
- **Total Students** - Enrolled students
- **Total Staff** - Active staff members
- **Active Files** - Files currently in circulation

### Quick Actions

- Create new files
- View pending tasks
- Access recent reports

## Schools Management

### View Schools

1. Click "Schools" in the sidebar
2. View list of all schools
3. Use filters to narrow down results

### Add School

1. Click "Add School" button
2. Fill in school details:
   - Name
   - Code
   - Type (Junior/Senior)
   - LGA
   - Address
3. Click "Create"

### Edit School

1. Click the edit icon on a school row
2. Modify details
3. Click "Update"

---

## Staff Management

### View Staff

1. Click "Staff" in the sidebar
2. View list of all staff members
3. Filter by category, designation, or school

### Add Staff

1. Click "Add Staff" button
2. Fill in staff details:
   - Personal information
   - Employment details
   - Bank information
3. Click "Create"

### Manage Leave

1. Navigate to Staff > Leaves
2. View leave requests
3. Approve or reject requests

---

## Student Management

### View Students

1. Click "Students" in the sidebar
2. View list of all students
3. Filter by school, class, or status

### Add Student

1. Click "Add Student" button
2. Fill in student details:
   - Personal information
   - Parent/guardian details
   - Medical information
3. Click "Create"

### View Student Profile

1. Click on a student name
2. View complete profile including:
   - Academic records
   - Attendance history
   - Medical records

## E-Registry

### Create File

1. Click "Registry" in the sidebar
2. Click "Create File"
3. Fill in file details:
   - Title
   - Type (Correspondence, Memo, etc.)
   - Classification
   - Priority
4. Click "Create"

### Track File Movement

1. Open a file
2. View movement history
3. See current holder
4. Track status changes

### Move File

1. Open the file
2. Click "Move File"
3. Select recipient
4. Add remarks
5. Click "Submit"

---

## Finance Management

### View Payments

1. Click "Finance" in the sidebar
2. View all payments
3. Filter by status, date, or student

### Record Payment

1. Click "Record Payment"
2. Select student and fee type
3. Enter payment details
4. Click "Submit"

### Generate Invoice

1. Select student
2. Choose fee structure
3. Generate invoice
4. Export as PDF

---

## Reports

### View Reports

1. Click "Reports" in the sidebar
2. Browse available reports by category

### Export Report

1. Select a report
2. Choose export format:
   - PDF
   - Excel
   - Word
   - JPEG
3. Click export button

### Import Data

1. Click "Import Data"
2. Select file type (Excel, CSV, PDF, Word)
3. Upload file
4. Review imported data
5. Confirm import

---

## Settings

### Profile Settings

1. Click your avatar in the header
2. Select "Profile"
3. Update your information
4. Click "Save"

### Change Password

1. Go to Settings > Security
2. Enter current password
3. Enter new password
4. Confirm new password
5. Click "Change Password"
